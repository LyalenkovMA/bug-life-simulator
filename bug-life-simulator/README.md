## 🎮 **О проекте**

**Tales from the Underbrush** — это симулятор жизни в мире антропоморфных насекомых, где каждый жук — уникальная личность со своими потребностями, характером и социальными связями. Игра сочетает элементы RPG, life-simulation и стратегии.

### **Ключевые особенности**
- **Глубокая система потребностей** по пирамиде Маслоу
- **Динамическая система состояний (FSM)** для AI персонажей
- **Социальная механика** с отношениями и репутацией
- **Профессиональный рост** через обучение у мастеров
- **Пороки и зависимости** как механика риска/награды
- **Процедурно генерируемый мир** с уникальными локациями

## 🚀 **Начало работы**

### **Требования**
- [.NET 8.0 SDK](https://dotnet.microsoft.com/download)
- [Visual Studio 2022](https://visualstudio.microsoft.com/) или [Rider](https://www.jetbrains.com/rider/)
- [MonoGame 3.8](https://www.monogame.net/downloads/)

### **Установка и запуск**
```bash
# 1. Клонируйте репозиторий
git clone https://github.com/yourusername/bug-life-simulator.git
cd bug-life-simulator

# 2. Восстановите зависимости
dotnet restore

# 3. Запустите проект
dotnet run --project src/BugLifeSim.csproj
```

Или откройте `BugLifeSim.sln` в Visual Studio и нажмите **F5**.

## 🎯 **Игровая механика**

### **Система благополучия (Wellness System)**
Персонажи имеют иерархию потребностей:
1. **Физиологические** (голод, жажда, сон)
2. **Безопасность и комфорт**
3. **Социальное признание**
4. **Самореализация**
5. **Предназначение** (эндгейм)

### **Развитие персонажа**
- **3 профессии на игру** — стратегический выбор
- **Обучение у мастеров** — поиск учителей в мире
- **Очки просветления** — открывают новые возможности
- **Пороки и зависимости** — рискованные пути к быстрому успеху

### **Мир и исследование**
- **Глобальная карта** с пошаговым перемещением
- **5 типов локаций** от безопасных поселений до диких земель
- **Точки интереса** с процедурными подземельями
- **Динамическая экономика** и система владений

## **Дополнительные разделы (опционально)**

### 🎨 **Художественный стиль**
Игра выполнена в стилизованной 2D-изометрии с акцентом на:
- **Антропоморфных насекомых** с уникальными чертами
- **Органическую архитектуру** (ульи, гнёзда, грибные дома)
- **Динамическое освещение** и погодные эффекты
- **Минималистичный UI**, встроенный в игровой мир

### 🎵 **Звуковое сопровождение**
- **Атмосферный саундтрек** с элементами природы
- **Динамическая система звуков** (насекомые, погода, деятельность)
- **Озвучка ключевых персонажей** (планируется)

### 📊 **Планы развития**
- [ ] MVP: "Жизнь одного жука" (1 локация, базовые системы)
- [ ] Альфа: 3 локации, социальная механика
- [ ] Бета: 5+ локаций, магия, профессии
- [ ] Релиз: Полный контент, Steam/itch.io публикация
